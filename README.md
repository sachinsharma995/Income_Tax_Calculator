# Income Tax Calculator

An **Income Tax Calculator** built using **HTML, CSS, and JavaScript** that calculates estimated tax based on yearly income. The project features a modern and responsive UI with instant results.

## Features

* Calculate tax based on entered annual income
* Clean and attractive user interface
* Fully responsive design
* Instant result display using JavaScript
* Beginner-friendly project for practicing DOM manipulation and logic building

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

* Enter your yearly income
* Click on **Calculate Tax**
* The calculator displays the total tax amount instantly based on tax slabs

## Purpose of Project

This project was created to improve JavaScript skills such as:

* Event Handling
* Form Validation
* Conditional Statements
* DOM Manipulation
